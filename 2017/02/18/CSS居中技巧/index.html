该方法的要点是给父元素设一个合适的 font-size的值，这个值的取值为该父元素的高度除以1.14得到的值，并且子元素必须 是一个 inline或 inline-block元素，需要加上 vertical-align: middle属性。使用这种方法，子元素的宽度或高度都不必知道。
